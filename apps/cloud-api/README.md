# Cloud API

Servico cloud proprio do Smart-Band.

Responsabilidades:

- identidade e cadastro dos sites/edges
- ingestao idempotente de transacoes sincronizadas
- consolidacao, reconciliacao e deteccao de divergencias
- comandos administrativos assincronos
- integracao isolada com a EasySmart Platform

Nao participa da autorizacao sincrona de uma atracao.
