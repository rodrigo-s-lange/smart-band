# Edge API

Autoridade transacional local do evento.

Responsabilidades:

- participantes, sessoes e associacao de pulseiras
- ledger, saldo, carga, debito, estorno e ajuste
- gateways, atracoes e politicas de acesso
- atomicidade, idempotencia, auditoria e autenticacao
- outbox duravel na mesma fronteira de consistencia

Nao depende da internet nem da cloud para autorizar uma transacao.
