# Appliance deployment

Instalação local em Raspberry Pi ou mini-PC Linux com SSD.

## Requisitos

- deploy reproduzível a partir do GitHub
- configuração e segredos externos
- PostgreSQL sem porta pública
- backend e frontend iniciados automaticamente
- navegador em modo kiosk
- operação integral sem internet
- backup criptografado e restore testado
- atualização assinada com rollback
- nenhuma dependência do laboratório i5

Serviços externos de licença, atualização, suporte e backup são opcionais e não
participam da autorização de atrações.
