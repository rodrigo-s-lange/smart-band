# Simulador de pulseira

Primeira implementação executável do protocolo.

Deve simular:

- pressão longa e geração da solicitação
- código OLED
- advertising efêmero
- desafio GATT
- exibição de atração e custo
- confirmação, timeout (10s sem clique) e recebimento de resultado `denied`
- AES-128-CMAC, contador, replay e saldo visual (ADR 0003)
